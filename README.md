== README

Install
-------

- ruby []()
- install Rails [](http://guides.rubyonrails.org/getting_started.html)
- install sqlite3 []()

- `gem install bundler` [](http://bundler.io/)
- `gem install guard` [](https://github.com/guard/guard)

Start
-----

- `bundler install`
- `bower install`
- `ruby bin\rails server`
- `bundle exec guard` *for live-reload*
